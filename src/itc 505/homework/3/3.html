Web font service

A web font is any font used in a website’s design that isn’t installed by default on the end user’s device—a counterpart to a system font.

One useful method to deliver custom fonts to html file is by using the CSS google font rule. Google Fonts makes it easy to bring personality and performance to your websites and products. Our robust catalog of open-source fonts and icons makes it easy to integrate expressive type and icons seamlessly — no matter where you are in the world.
